Comandos para levantar el proyecto

Ejecuta estos comandos desde la raiz del proyecto. 

npm install
npm run build:web
npm run serve:web
# Abrir: http://localhost:5500/src/index.html

# Desarrollo (2 terminales)
npm run watch:web
npm run serve:web

# Ejecutar un ejercicio TS por consola
npm run ts -- 06-integracion-api.ts
